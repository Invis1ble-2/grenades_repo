# 道具包文件目录

## 目录结构

```
packages/
├── dust2/           # Dust2 地图道具包
│   └── xxx.cs2pkg
├── inferno/         # Inferno 地图道具包
│   └── xxx.cs2pkg
├── mirage/          # Mirage 地图道具包
│   └── xxx.cs2pkg
├── anubis/          # Anubis 地图道具包
│   └── xxx.cs2pkg
├── ancient/         # Ancient 地图道具包
│   └── xxx.cs2pkg
├── nuke/            # Nuke 地图道具包
│   └── xxx.cs2pkg
├── vertigo/         # Vertigo 地图道具包
│   └── xxx.cs2pkg
└── xxx_全图.cs2pkg  # 全图整合包直接放这里
```

## 命名规范

- **单地图包**: `packages/地图名/作者名_描述.cs2pkg`
  - 例如: `packages/dust2/player123_smokes.cs2pkg`
- **全图整合包**: `packages/作者名_全图整合.cs2pkg`
  - 例如: `packages/official_全图精选.cs2pkg`
